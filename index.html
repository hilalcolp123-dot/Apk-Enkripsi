Teks Enkripsi
Panduan Penggunaan & Detail Fitur
1
Cara Enkripsi (Mengunci Pesan)
  • Pastikan tombol utama berwarna biru bertuliskan "Enkripsi".
  • Ketik pesan rahasia Anda pada kolom Teks Asli.
  • Masukkan password pada kolom Kunci.
  • Klik tombol Enkripsi. Kode acak akan muncul di kolom kanan.
2
Cara Dekripsi (Membuka Pesan)
  • Klik tombol untuk pindah ke mode Dekripsi (Tombol Hijau).
  • Tempel kode acak (Ciphertext) di kolom Teks Terenkripsi.
  • Masukkan password yang digunakan saat mengunci tadi.
  • Klik tombol Dekripsi untuk melihat pesan aslinya.
Mengenal Fungsi Tombol
Ganti Mode (Toggle) Berfungsi menukar posisi antara mode Enkripsi (mengunci) dan Dekripsi (membuka).
Salin Teks Menyalin hasil enkripsi/dekripsi ke clipboard agar mudah dikirim (WA/Email).
Bersihkan Form Menghapus semua inputan dan mereset kunci. Gunakan ini sebelum membuat pesan baru.
Format Huruf Otomatis mengubah teks menjadi HURUF KAPITAL atau huruf kecil jika diinginkan.
Catatan Keamanan Penting:

Algoritma AES-256 bersifat satu arah tanpa celah. Jika Anda lupa password atau salah memasukkan kunci (meskipun hanya kurang satu spasi), pesan tidak akan pernah bisa dibuka kembali.
